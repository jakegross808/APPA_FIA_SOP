Description:
Abstract:
This data set includes polygons for ecological sections and subsections within Subregions within the conterminous United States clipped to the HUC10 shell surrounding the Appalachian National Scenic Trail. This data set contains regional geographic delineations for analysis of ecological relationships across ecological units.
Purpose:
Baseline map of the ecological boundaries. Ecological analysis of resource distribution patterns and associations among different ecological land units.
